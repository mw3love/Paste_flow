# 행번호 사각 테두리 박스 제거 수정계획

## 문제 현상
- 패널의 각 클립보드 항목 좌측 행번호(1, 2, 3, 4…) 주위에 **악센트 색상(주황/청록/빨강)의 사각 테두리 박스**가 그려짐
- 항목 상태별 색상이 그대로 반영됨: 일반=주황, 현재 큐=청록, 고정=빨강
- 재현 조건: **항상** 발생 (패널을 열면 무조건 보임)
- 여러 차례 수정 요청했으나 개선되지 않음 → 근본 원인 미파악 상태

## 원인 분석

### 1순위: Qt 스타일시트 하위 위젯 상속 문제 (핵심 원인)

- 파일: `pasteflow/ui/panel.py` L211~L218 (`_apply_bg_style`)
- `PanelItemWidget.setStyleSheet(...)` 호출 시 `border: 1px solid {accent_color};` 가 포함됨
- Qt 스타일시트는 **부모 QWidget의 border/background 속성이 자식 QLabel에 cascade**됨
- `num_label` (L110): stylesheet에 `border:` 명시 없음 → 부모의 `border: 1px solid {color}` 상속
- 결과: 22×26px 크기의 num_label에 악센트 색 테두리가 나타나 "사각 박스"처럼 보임

```python
# L211~L218 — 부모 위젯에 border 설정 → 자식 QLabel로 cascade
def _apply_bg_style(self):
    border_color = self._accent_color
    # 아래 border 속성이 num_label, text_label, thumb_label에 cascade됨
    self.setStyleSheet(f"background-color: {COLORS['surface0']}; border-radius: 6px; border: 1px solid {border_color};")
```

```python
# L107~L110 — num_label에 border: none 없음 → 부모 border 상속
num_label = QLabel(str(number))
num_label.setStyleSheet("color: #ffffff; font-size: 11px; background: transparent;")
#                                                                     ↑ border: none 누락
```

### 2순위: text_label, thumb_label도 동일 문제 잠재

- `text_label` (L139): `stylesheet`에 `border:` 없음 → 넓은 라벨이라 눈에 덜 띄지만 실제로는 border 적용됨
- `thumb_label` (L116~L122): `setStyleSheet` 자체가 없음 → border 그대로 cascade

### del_btn이 안 보이는 이유

- `del_btn` (L163~L175): 자체 stylesheet에 `border: none;` 명시되어 있어 부모 cascade 차단됨
- num_label은 이 처리가 누락된 것

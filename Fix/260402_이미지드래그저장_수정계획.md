# 이미지 드래그 → 파일 저장 기능 수정계획

## 기능 설명
- 패널에서 이미지 항목을 바탕화면 또는 탐색기 폴더로 드래그하면 PNG 파일로 저장
- 텍스트 항목은 기존 붙여넣기 동작 유지

## 현상 및 목표

| 상황 | 현재 | 목표 |
|------|------|------|
| 이미지 → Explorer 폴더 드래그 | WM_PASTE 전송 (무효) | PNG 파일로 저장 |
| 이미지 → 바탕화면 드래그 | WM_PASTE 전송 (무효) | PNG 파일로 저장 |
| 텍스트 → 외부 앱 드래그 | WM_PASTE / SendInput | 변경 없음 |
| 이미지 → 일반 앱 드래그 | WM_PASTE / SendInput | 변경 없음 (클립보드로 붙여넣기) |

## 원인 분석

### 현재 드래그 아키텍처 (fake drag)

```
mouseMoveEvent → 커서만 DragCopyCursor로 변경 (실제 OLE 없음)
mouseReleaseEvent → external_drag_paste.emit(item_id, cursor_pos)
  → ClipboardPanel.drag_to_app_requested.emit
    → PasteFlowApp._on_drag_to_app(item_id, cursor_pos)
      → _set_clipboard(item) → WM_PASTE 또는 SendInput(Ctrl+V)
```

Explorer/바탕화면은 OLE 드롭 타겟이므로 `WM_PASTE`나 `SendInput`으로 파일을 받을 수 없음.

### 1순위: 대상 창 분류 부재
- 파일: `pasteflow/main.py` `_on_drag_to_app()` L349-381
- Explorer(`CabinetWClass`)와 바탕화면(`Progman`, `WorkerW`)을 별도 처리하는 코드 없음
- 이미지 항목인지 확인하는 분기도 없음

### 2순위: 이미지→PNG 변환 로직 부재
- `image_data`는 PNG bytes (`\x89PNG...`) 또는 CF_DIB raw bytes로 저장
- DIB → PNG 변환 시 BMP 파일 헤더를 수동으로 조립해야 함

## 설계 결정

### OLE DoDragDrop 대신 mouseRelease 시점 파일 저장 방식 채택

- **이유**: 현재 fake drag 아키텍처 유지, 새 COM 의존성 없음
- **방식**: mouseRelease → 대상 창 클래스 확인 → Explorer/Desktop이면 폴더 경로 취득 → PNG 저장
- **Explorer 폴더 경로 취득**: `win32com.client` (pywin32 내장)의 `Shell.Application` IShellWindows

### Explorer 창 분류 기준

| 창 클래스명 | 대상 | 처리 |
|-------------|------|------|
| `CabinetWClass` | 탐색기 폴더 창 | `Shell.Application`으로 현재 폴더 경로 취득 |
| `Progman` | 바탕화면(레거시) | `%USERPROFILE%\Desktop` |
| `WorkerW` | 바탕화면(Win10/11) | `%USERPROFILE%\Desktop` |

### 이미지 포맷 처리

| `image_data` 시작 | 포맷 | 저장 방식 |
|-------------------|------|----------|
| `b'\x89PNG'` | PNG bytes | 직접 `.write()` |
| 그 외 | CF_DIB raw | BMP 헤더 조립 후 Pillow로 PNG 변환 |

### 파일명

```
clip_YYYYMMDD_HHMMSS.png
예) clip_20260402_153045.png
```

충돌 방지: 같은 초에 저장 시 `_1`, `_2` suffix 추가.

### Phase 1 — Explorer/바탕화면 감지 + 이미지 파일 저장

**변경 위치**: `pasteflow/main.py`

**추가할 헬퍼 함수 (모듈 레벨)**:

```python
_EXPLORER_CLASSES = {"CabinetWClass"}
_DESKTOP_CLASSES = {"Progman", "WorkerW"}

def _get_explorer_folder(hwnd: int) -> Optional[str]:
    """CabinetWClass hwnd → 현재 폴더 경로. 실패 시 None."""
    try:
        import win32com.client
        shell = win32com.client.Dispatch("Shell.Application")
        for window in shell.Windows():
            try:
                if int(window.HWND) == hwnd:
                    return window.Document.Folder.Self.Path
            except Exception:
                continue
    except Exception:
        pass
    return None

def _get_desktop_path() -> str:
    try:
        import win32com.client
        return win32com.client.Dispatch("WScript.Shell").SpecialFolders("Desktop")
    except Exception:
        return os.path.expanduser("~/Desktop")

def _save_image_to_folder(image_data: bytes, folder: str) -> str:
    """image_data(PNG or DIB) → folder에 PNG 파일 저장. 저장 경로 반환."""
    import io, struct, os
    from PIL import Image
    from datetime import datetime

    ts = datetime.now().strftime("%Y%m%d_%H%M%S")
    base = os.path.join(folder, f"clip_{ts}.png")
    # 충돌 방지
    path = base
    suffix = 0
    while os.path.exists(path):
        suffix += 1
        path = os.path.join(folder, f"clip_{ts}_{suffix}.png")

    if image_data[:4] == b'\x89PNG':
        with open(path, 'wb') as f:
            f.write(image_data)
    else:
        # CF_DIB → BMP 파일 헤더 조립 → Pillow PNG 변환
        bih_size = struct.unpack_from('<I', image_data, 0)[0]
        bit_count = struct.unpack_from('<H', image_data, 14)[0]
        colors_used = struct.unpack_from('<I', image_data, 32)[0]
        if bit_count <= 8:
            n_colors = colors_used if colors_used > 0 else (1 << bit_count)
        else:
            n_colors = colors_used
        pixel_offset = 14 + bih_size + n_colors * 4
        file_size = 14 + len(image_data)
        file_header = struct.pack('<2sIHHI', b'BM', file_size, 0, 0, pixel_offset)
        img = Image.open(io.BytesIO(file_header + image_data))
        img.save(path, 'PNG')

    return path
```

**`_on_drag_to_app` 수정 — 이미지 항목 + Explorer/Desktop 분기**:

```python
def _on_drag_to_app(self, item_id: int, cursor_pos):
    full_item = self.db.get_item(item_id)
    if not full_item:
        return

    import win32gui, win32con

    screen_pt = (cursor_pos.x(), cursor_pos.y())
    hwnd = win32gui.WindowFromPoint(screen_pt)
    if not hwnd:
        return

    # 최상위 창 클래스 확인
    root_hwnd = win32gui.GetAncestor(hwnd, win32con.GA_ROOT)
    root_class = ""
    try:
        root_class = win32gui.GetClassName(root_hwnd)
    except Exception:
        pass

    # 이미지 항목 → Explorer/바탕화면이면 파일 저장
    if full_item.image_data and full_item.content_type == "image":
        folder = None
        if root_class in _EXPLORER_CLASSES:
            folder = _get_explorer_folder(root_hwnd)
        elif root_class in _DESKTOP_CLASSES:
            folder = _get_desktop_path()
        if folder:
            _save_image_to_folder(full_item.image_data, folder)
            return  # 파일 저장 완료 — 클립보드 붙여넣기 불필요

    # 기존 붙여넣기 경로
    self.interceptor._set_clipboard(full_item)
    target = _find_deepest_child(hwnd, screen_pt)
    class_name = ""
    try:
        class_name = win32gui.GetClassName(target)
    except Exception:
        pass
    if _is_chromium_window(class_name):
        top_hwnd = win32gui.GetAncestor(target, win32con.GA_ROOT)
        _activate_and_send_ctrl_v(top_hwnd)
    else:
        win32gui.SendMessage(target, win32con.WM_PASTE, 0, 0)
```

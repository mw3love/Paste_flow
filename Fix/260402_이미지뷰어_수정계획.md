# 이미지 미리보기 기능 확장 수정계획

## 문제 현상
- 패널에서 이미지 항목 더블클릭 시 미리보기 팝업이 열림
- 현재 팝업은 위치 고정, 크기 고정, 단일 창만 가능
- 다음 기능이 없음:
  1. 미리보기 창 드래그하여 이동
  2. 휠로 이미지 확대/축소
  3. ESC로 미리보기 닫기
  4. 여러 이미지 동시 미리보기

- 재현 조건: 항상 (현재 기능 자체가 없음)

---

## 원인 분석

### 1순위: ToolTip 윈도우 타입으로 인한 상호작용 불가
- 파일: `pasteflow/ui/image_preview.py` L31~35
- `Qt.WindowType.ToolTip` 플래그는 키보드 포커스 및 마우스 드래그 이벤트를 받지 않음
- ESC, 드래그, 휠 이벤트가 전달되지 않아 모든 상호작용 기능이 불가능함
- 수정: `Tool | FramelessWindowHint | WindowStaysOnTopHint`으로 변경

### 2순위: 싱글톤 패턴으로 인한 단일 인스턴스 제한
- 파일: `pasteflow/ui/image_preview.py` L21~27, L78~83
- `_instance` 클래스 변수와 `toggle_preview`가 하나의 창만 허용
- 여러 이미지 동시 미리보기를 위해선 인스턴스 목록으로 변경 필요
- `main.py` L568~571의 `_on_preview_image`도 함께 수정 필요

## Phase 실행 프로토콜

> **이 계획서를 열고 Phase를 구현하는 Claude는 아래 프로토콜을 반드시 따른다.**
>
> 1. 각 Phase의 모든 코드 수정 및 커밋이 완료되면
> 2. **eval-plan 서브에이전트를 포그라운드로 즉시 실행한다** — `evaluate` 모드, 커밋 범위는 해당 Phase의 커밋 수만큼 (`HEAD~N`)
> 3. 평가 결과를 사용자에게 보고한 후 다음 Phase 진행 여부를 확인한다
> 4. Critical/High 발견 시 — 다음 Phase 진행 전에 수정 여부를 사용자에게 확인한다
> 5. 각 작업 항목 완료 즉시 체크박스를 `[x]`로 업데이트한다 — 중단 후 재개 시 진행 상태 파악용

## 단계별 수정 계획

> 각 Phase는 새로운 대화세션에서 시작한다.
> 중단되어도 다음 세션에서 이 파일을 열어 이어서 진행 가능.

### Phase 1 — 드래그 이동 + 닫기 버튼 + 윈도우 타입 변경
> C-1/C-2 해소: `WA_ShowWithoutActivating` 유지 → 패널 포커스 보호. 창 클릭 시 포커스 자연 획득 → 이후 ESC 동작.
> ESC 불가 시 닫기 수단으로 × 버튼을 Phase 1에서 함께 구현.
- [x] `image_preview.py`: 윈도우 플래그 `ToolTip` → `Tool | FramelessWindowHint | WindowStaysOnTopHint`
- [x] `image_preview.py`: `WA_ShowWithoutActivating` **유지** (패널 포커스 탈취 방지)
- [x] `image_preview.py`: `mousePressEvent` — `event.button() == LeftButton`이면 드래그 시작점 저장
- [x] `image_preview.py`: `mouseMoveEvent` — 드래그 델타만큼 창 이동 (`self.move(...)`)
- [x] `image_preview.py`: `keyPressEvent` — `Key_Escape`이면 `self.close()` (창 클릭 후 동작)
- [x] `image_preview.py`: 우상단 × 닫기 버튼 추가 (`QToolButton`, 항상 표시)
- [x] `image_preview.py`: 최대 표시 크기 상한 320×240 → 640×480으로 변경 (`PREVIEW_MAX_W/H`)
- [x] 커밋: `Feat: 이미지 미리보기 드래그 이동·닫기 버튼 추가`
- [x] eval-plan evaluate 자동 실행

### Phase 2 — 휠 줌
- [x] `image_preview.py`: `_original_pixmap` 인스턴스 변수 추가 (원본 픽스맵 보존)
- [x] `image_preview.py`: `_scale_factor: float = 1.0` 인스턴스 변수 추가
- [x] `image_preview.py`: `show_preview()`에서 `self._original_pixmap = pixmap` (스케일 전 원본 저장)
- [x] `image_preview.py`: `wheelEvent` — 휠 업 +10%, 휠 다운 -10%, 범위 0.1×~8.0×, `_apply_scale()` 호출
- [x] `image_preview.py`: `_apply_scale()` 헬퍼 — `_original_pixmap`에 현재 배율 적용 후 `setPixmap` + `adjustSize` + 화면 경계 클램핑 (H-3 해소)
- [ ] 커밋: `Feat: 이미지 미리보기 휠 줌 추가`
- [ ] eval-plan evaluate 자동 실행

### Phase 3 — 여러 이미지 동시 미리보기
- [ ] `image_preview.py`: 싱글톤 `_instance` → `_instances: list[ImagePreviewPopup]` 클래스 변수로 교체
- [ ] `image_preview.py`: `open_new(cls, image_data, global_pos)` 클래스메서드 — 새 인스턴스 생성, `_instances`에 등록

# 260325_클릭큐선택_더블클릭미리보기

## 목표
- 단일 클릭: 미리보기 → **큐 시작점 선택** (해당 항목~최신까지 순차 붙여넣기)
- 더블클릭: 직접 붙여넣기 → **미리보기 팝업 토글**
- 직접 붙여넣기: 우클릭 메뉴로만 접근

## 확정 동작 스펙
| 액션 | 히스토리 항목 | 핀 항목 |
|------|-------------|---------|
| 단일 클릭 | 해당 항목부터 최신까지 큐 설정 (`queue_select_requested`) | 기존 유지: 클립보드 복사 + 큐 추가 |
| 더블 클릭 | 미리보기 팝업 토글 | 미리보기 팝업 토글 |
| 우클릭 | 붙여넣기 메뉴 유지 | 붙여넣기 메뉴 유지 |
| 큐 선택 후 패널 | 패널 유지 (자동 닫힘 없음) | — |

## 큐 구성 로직
DB는 newest-first (ORDER BY id DESC) 반환.
선택한 item_id가 index `i` 에 있을 때:
```python
history = db.get_recent_items()        # [E, D, C, B, A] (E가 최신)
queue_items = list(reversed(history[0:i+1]))  # 클릭 C(i=2) → [C, D, E]
queue.set_queue(queue_items)
```
Ctrl+V 순서: C → D → E
